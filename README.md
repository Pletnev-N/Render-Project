# Project description

This project is a C++ DirectX 11 rendering application created primarily for learning DirectX 11 API. The overall architecture and optimization are not well developed yet, as the main focus is currently on deepening graphics programming skills and exploring the DX11 rendering pipeline.

# Current state

### What has been done so far
* Reading and rendering mesh data from FBX file
* Phong shading
* Reading textures from TGA file
* Rendering diffuse map
